Definitions

How to define a table

<table> defines a table. Use elements listed below to specify content inside specific cells.

When to use:

<tr>: Table row. Use when starting a row.

<td>: Table data. Use when writing information in a cell of a table.

<th>: Table head. Use when defining a cell as a header.

<thead>: Use when grouping headers in a table.

<tfoot>: Use when grouping footers in a table.

<caption>: Adding a title or caption to a table.

Difference between colspan and rowspan.

Colspan combines cells across a table (horizontally) and rowspan combines cells vertically.

Difference between cell spacing and padding.

Cell spacing is the space between two cells in a table, and padding is the space between the border of a cell and the content in it.

How to define table borders with CSS.

In a CSS file, use border property to specify table borders. Define line weight and solid or dashed line.

How to control border widths.

Use border-width property to define thickness of borders. Specify border width:thin/medium/thick.
