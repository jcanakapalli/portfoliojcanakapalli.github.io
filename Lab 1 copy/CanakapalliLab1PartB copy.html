Terms and Definitions

DOCTYPE statement
A declaration that defines what type of document it is.
How to use: <ol>,<ul>, & <dl>
Use <ol> for ordered lists, <ul> for unordered lists, and <dl> for description lists. For <ol> and <ul>, follow the element with <li> for the list content. For <dl>, use <dt> for the term and <dd> for the definition.
Difference between text editor and word processor
A text editor is used to create and edit text files, while a word processor also formats documents.
Client-server model
A program sends a request to another program and waits for a response.
FTP
File transfer protocol. Used to transfer files on the internet but is insecure.
HTTP
Hypertext transfer protocol. Transmits documents from the internet.
TCP/IP
Transmission control protocol/internet protocol. Allows two hosts to connect and exchange data.
